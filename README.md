# EduTrack Dashboard

**EduTrack Dashboard** is a real‑time student management system built with **Python (Streamlit + Pandas)**. It helps track attendance and marks, visualize performance, and generate alerts for low attendance. The app supports adding/updating/deleting students and exporting data to CSV/Excel.

---

## 🚀 Features
- 📊 Interactive charts for marks and attendance trends  
- ✅ Alerts for students with attendance below 75%  
- ✏️ Add, update, or delete student records directly  
- 📥 Export updated data as CSV/Excel  
- 🌐 Deployed on Streamlit Cloud for easy access  

---

## 🛠 Tech Stack
- **Python**  
- **Streamlit** for UI  
- **Pandas** for data handling  
- **XlsxWriter** for Excel export
